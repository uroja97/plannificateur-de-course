# Planificateur de Courses et Repas : Famille Anti-inflammatoire 🥑🛒

Bienvenue dans l'espace de travail du **Planificateur de Courses**.

L'agent Antigravity est prêt à vous aider à générer un menu de 2 semaines et votre liste de courses, en respectant un profil sans inflammation, et adapté pour vos achats chez Costco pour 4 personnes (~400$).

## Structure
- `.agent/workflows/planifier_courses.md` : Guide étape par étape d'Antigravity (tapez `/planifier_courses` pour l'utiliser !).
- `.agent/skills/planificateur_anti_inflammatoire/SKILL.md` : Consignes (diète, enfants, budget, Costco).
- `base_de_donnees/produits_frequents_costco.md` : Tableau qui sera enrichi de vos produits et de leurs prix au fil de vos ajouts de tickets de caisse.
- `historique_plannings/` : Dossier où vos anciens menus et listes de courses seront archivés.

Pour commencer, tapez : `/planifier_courses` dans la console d'Antigravity !
